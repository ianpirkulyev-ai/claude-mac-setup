# Текст для пересылки в Telegram

Скопируй блок ниже (между линиями) и перешли человеку в Telegram.

---

Привет! Сейчас настроим тебе компьютер - голосовой ввод (диктовка вместо клавиатуры),
расшифровка аудио/видео в текст и удобная система заметок. Делать почти ничего не надо:
большую часть сделает ИИ-помощник. От тебя - 4 простых шага.

1. Установи редактор VS Code:
   https://code.visualstudio.com - нажми синюю кнопку Download, открой скачанный файл, перетащи в Программы.

2. Установи помощника Claude внутри VS Code:
   - открой VS Code
   - слева на панели нажми значок из квадратиков (Extensions / Расширения)
   - в строке поиска набери: Claude Code
   - у расширения от Anthropic нажми Install

3. Войди в Claude:
   - после установки слева появится значок Claude - нажми его
   - нажми кнопку входа (Sign in) и залогинься своим аккаунтом Claude
     (нужна подписка Claude; если её нет - напиши мне, помогу оформить)

4. Запусти настройку:
   - создай на рабочем столе пустую папку (назови, например, setup) и открой её в VS Code
     (File - Open Folder)
   - открой окно чата Claude и вставь туда вот это сообщение:

   ----- скопируй отсюда -----
   Привет! Склонируй репозиторий https://github.com/ianpirkulyev-ai/claude-mac-setup,
   открой в нём файл SETUP-INSTRUCTION.md и выполни все шаги по порядку, проводя меня
   за руку. Я не программист - объясняй простыми словами и говори, что нажимать.
   Если попросишь разрешение запустить команду - я буду нажимать Allow. Начни сейчас.
   ----- досюда -----

Дальше просто следуй подсказкам Claude и нажимай Allow, когда он просит разрешение.
Если что-то непонятно - пришли мне скриншот.

---

## Подсказки для тебя (НЕ пересылать)

- Всё работает на Mac. Для Windows шаги те же, но команды установки Whisper отличаются -
  Claude разберётся по `SETUP-INSTRUCTION.md` (там есть ветки под разные системы; чисто
  Windows-сценарий в инструкции пока не расписан - при необходимости дополним).
- «Подписка Claude» = аккаунт claude.ai (Pro или выше) либо вход через рабочий аккаунт.
  Без входа помощник не запустится.
- Если у человека уже стоит VS Code и расширение - он сразу делает шаг 4.
- Весь комплект и сама инструкция: https://github.com/ianpirkulyev-ai/claude-mac-setup

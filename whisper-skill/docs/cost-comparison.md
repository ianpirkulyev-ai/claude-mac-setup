# Local vs OpenAI API — сколько ты экономишь

OpenAI Whisper API — **$0.006 / минута** аудио. Локальный Whisper — **$0** после первого запуска.

## Расчёты для типовых сценариев

### Сценарий 1 — TikTok-аналитик (находишь виралки, переписываешь)

```
50 видео в день × 1 минута среднее = 50 мин/день
50 × 30 дней = 1500 мин/мес

OpenAI:    1500 × $0.006 = $9/мес
Локально:  $0

За год: $108 экономии
```

### Сценарий 2 — Подкастер

```
4 эпизода в неделю × 60 минут = 240 мин/неделю
240 × 4.3 недели = ~1000 мин/мес

OpenAI:    1000 × $0.006 = $6/мес
Локально:  $0

За год: $72
```

⚠️ Тут есть второй слой: подкасту нужна diarization. OpenAI Whisper API **не делает diarization вообще**. Так что локально через whisperx — единственный путь.

### Сценарий 3 — Контент-creator full-time

```
Транскрибируешь видео конкурентов + свои подкасты + интервью гостей + диктовка:
~5 часов аудио в день = 300 мин/день
300 × 30 = 9000 мин/мес

OpenAI:    9000 × $0.006 = $54/мес
Локально:  $0

За год: $648
```

### Сценарий 4 — Агентство (10 клиентов)

10 клиентов × ~30 часов аудио в месяц = 18000 мин/мес

OpenAI:    18000 × $0.006 = $108/мес
Локально:  $0

### Сценарий 5 — Студия дубляжа / локализации

50 часов аудио в неделю × 4 = 12000 мин/мес

## Стоимость "локального"

«Бесплатно» условное:

### Время на установку

- Mac M-чип: **5 минут** (`brew install ffmpeg && pip install mlx-whisper`)
- Linux NVIDIA: **30 минут** (CUDA Toolkit может быть капризным)
- Windows: **20 минут** через WSL2; час нативно
- Без GPU: **5 минут** (faster-whisper в int8)

### Дисковое место

- Одна модель `large-v3-turbo`: 1.6 GB
- Если ставишь diarization: + 0.5 GB
- Итого: 2-3 GB
- Удалить: `huggingface-cli delete-cache`

### Электричество

- На Mac M2 Pro 1 час транскрибации = 30-50 ватт-часов = **~$0.005**
- На Linux + RTX 4090 1 час транскрибации = 250-350 Вт × 6 мин = ~30 ватт-часов = **~$0.003**
- На CPU x86 1 час = ~150 ватт-часов = **~$0.015**

Совершенно несущественно.

## Когда API всё-таки выгоднее

⚠️ **Honest take** — есть случаи когда OpenAI API имеет смысл:

1. **Серверный пайплайн без своего железа** — если ты на Vercel/Cloudflare Workers и нет места для модели → API логичнее
2. **Очень редкие случаи** (раз в неделю файл) — установка не отобьётся
3. **Гарантированный SLA** — OpenAI держит uptime, твой ноут — нет
4. **Нет ноута / нет рук** — кому-то проще $5/мес чем разбираться с pip

Во всех остальных кейсах локальный Whisper выгоднее.

## Скрытая ценность локального

Помимо денег:

- **Приватность** — записи интервью / подкастов не уходят в OpenAI. Для медицины / юристов / финансов — **обязательно**.
- **Нет лимитов** — обработай хоть 1000 часов в день. На API — упрёшься в rate limit.
- **Тонкая настройка** — VAD, температура, prompts, post-processing — всё под твоим контролем
- **Diarization** — на API её просто нет
- **Word-level timestamps** — на API доступны через дорогой `verbose_json` с ограничениями. Локально — без проблем.
- **Multi-language fluency** — локально можешь брать `large-v3` для редких языков; на API один tier
- **Offline** — работает в самолёте / на даче / без интернета

## Альтернативные «дешёвые» API

Если совсем не хочется ставить локально:

- **Groq Whisper** (https://groq.com/) — Whisper-large-v3 на их LPU чипах. **Бесплатный tier** есть. Платный ~$0.00115/мин (в 5 раз дешевле OpenAI). Но всё равно платно.
- **Replicate** — Whisper модели через API, ~$0.001/мин
- **Hugging Face Inference Endpoints** — деплоишь свой Whisper на их серверах, оплата за время работы

Эти варианты — middle ground. Дешевле OpenAI, проще локального. Но всё равно не «$0».

## Вывод

Если ты регулярно работаешь с речью — **локально Whisper это no-brainer**:
- Mac M-чип у тебя? Установка 5 минут, скорость 17x realtime.
- Слабый ноут? `whisper.cpp + small` модель. Медленнее, но всё ещё бесплатно.
- Сервер с GPU? `faster-whisper + large-v3-turbo`. Промышленный уровень.

При любом сценарии экономия от $50 до $1500 в год + бонусы (приватность, diarization, offline).
